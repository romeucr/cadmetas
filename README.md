# cadMetas

## About
The project consists of: 
* Node.js backend API.
* Web page where you can list and create new items in a TODO list.
* Mobile app where you can list and create new items in a TODO list.

## Some technologies used:
### Back end
- Node.js
- Javascript

### Front-end (web and mobile)
- HTML / CSS / JS
- ReactJS
- React Native


### Database
 - MongoDB
